# Nasa Api Pictures

Get a random picture from NASA API

Check the demo hosted on https://filipeqs.github.io/note-taker/

---

## Instructions

First clone this repository

```bash
$ git clone https://github.com/filipeqs/note-taker.git
```

Open index.html

```bash
open index.html
```

## Features

- Load pictures
- Save to Favorites
- Show Favorites
- Delete Favorites
